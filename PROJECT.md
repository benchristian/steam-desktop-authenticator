# Steam 桌面令牌验证器 — 项目文档

> **版本**: v2.0.0  
> **平台**: macOS（兼容 Windows/Linux）  
> **技术栈**: Electron + Node.js + HTML/CSS/JS

---

## 一、项目概述

一个功能完备的 **Steam 桌面令牌验证器**，专为 macOS 设计。核心能力是通过 `steam-session` 模拟 Steam 手机 App 登录，直接绑定验证器，无需手动提取密钥。同时支持 TOTP 验证码生成、QR 扫码登录、交易确认、多账号管理等高级功能。

## 四、主要功能模块

### 4.1 验证码生成与显示

- **TOTP 算法**：与 Steam 手机 App 完全兼容的 5 位验证码
- **30 秒自动刷新**：带倒计时进度条
- **颜色变化**：绿色（正常）→ 黄色（即将过期）→ 红色（已过期）
- **大字体显示**：40px 等宽字体，清晰醒目
- **一键复制**：点击复制按钮即可复制验证码到剪贴板

### 4.2 一键绑定令牌

通过 `steam-session` 模拟手机 App 登录，四步完成令牌绑定：

1. **Steam 登录** — 输入用户名密码，支持 Steam Guard 验证码
2. **邮箱验证码** — 发送验证码到注册邮箱，提交后与 Steam 服务器时间同步
3. **撤销代码** — 显示 revocation_code，用户确认已保存
4. **完成** — 自动保存 maFile 文件

### 4.3 QR 扫码登录

最具特色的功能，无需手机即可完成扫码登录：

1. 用户在浏览器打开 Steam 登录页，显示二维码
2. 工具截取屏幕，用 `jsQR` 识别二维码位置
3. 自动模拟鼠标点击二维码
4. 使用已有令牌的 `LoginApprover` 批准登录

### 4.4 交易确认

- 查看待确认的 Steam 交易列表
- 支持单个确认/取消
- 支持一键全部确认
- 自动使用当前选中的账号

### 4.5 多账号管理

- **虚拟滚动**：50+ 账号时自动启用，流畅不卡顿
- **搜索筛选**：实时过滤，支持账号名和 SteamID 模糊匹配
- **右键菜单**：复制信息、导出文件、库存公开、交易保护确认、更新令牌等
- **导入导出**：支持 maFile JSON 格式导入导出

### 4.6 系统集成

- **系统托盘**：关闭窗口后最小化到托盘，托盘菜单可快速复制验证码
- **macOS 原生风格**：隐藏标题栏，红绿灯按钮区域，毛玻璃弹窗
- **快捷键**：`Esc` 关闭弹窗，`Cmd+N` 添加账号，`Cmd+F` 搜索

## 六、数据存储

- **位置**：`app.getPath('userData')/maFiles/`
- **格式**：每个账号一个 `.maFile` JSON 文件
- **关键字段**：
  - `account_name` — 账号名
  - `SteamID` — Steam ID（字符串格式避免大整数精度丢失）
  - `shared_secret` — 共享密钥（TOTP）
  - `identity_secret` — 身份密钥（交易确认）
  - `access_token` — JWT 访问令牌
  - `revocation_code` — 撤销代码
  - `web_cookies` — Steam 网页 Cookie

## 七、安全设计

- **上下文隔离**：`contextIsolation: true`，`nodeIntegration: false`
- **preload 桥接**：通过 `contextBridge` 暴露安全 API
- **本地存储**：所有数据存储在本地，不上传
- **代理支持**：支持 HTTP/SOCKS 代理隧道
